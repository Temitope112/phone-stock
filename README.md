# 📱 PhoneStock

PhoneStock is a modern SaaS platform built for independent phone vendors to manage inventory, record sales, store customer information, and showcase available phones through a public marketplace.

The platform provides vendors with a dedicated dashboard while allowing buyers to browse available phones and contact sellers directly via WhatsApp.

---

## 🚀 Live Demo

https://phone-stock.vercel.app

---

## ✨ Features

### Vendor Dashboard

- Secure authentication
- Phone inventory management
- Product image uploads
- Record sales
- Customer management
- Search and filter inventory
- Automatic stock updates
- Sales history
- Dashboard analytics

### Public Marketplace

- Browse available phones
- Search by phone, brand, model or vendor
- Filter by brand
- View phone specifications
- WhatsApp integration
- Marketplace disclaimer

### Admin Dashboard

- Platform overview
- Revenue tracking
- Vendor management
- Marketplace management
- Platform settings
- Registration control
- Recent sales
- Notifications

---

## 🔒 Security

PhoneStock is built with security in mind.

- Vendor data isolation using Supabase Row Level Security (RLS)
- Protected admin API routes
- Server-side authentication
- Secure password reset
- Protected database operations
- Server-only service role usage

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/Temitope112/phone-stock.git
```

Navigate into the project

```bash
cd phone-stock
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
